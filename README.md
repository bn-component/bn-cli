# bn-cli 

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x, 5.x preferred) and [Git](https://git-scm.com/).

``` bash
$ npm install -g bn-cli
```

### Usage

``` bash
$ bn init <git-path> <project-name>
```

Example:

``` bash
$ bn init xxxxx my-project
```
